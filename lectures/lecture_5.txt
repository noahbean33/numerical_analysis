## **Lecture Notes: Polynomial Interpolation (Part 1)**

**Topic:** Approximating continuous functions through discrete data points.


**Context:** This lecture explores alternatives to Taylor series, which, while useful, often fail in practical data scenarios where derivatives are unavailable.

---

### **1. The Interpolation Problem**

Unlike Taylor polynomials, which approximate a function at a single point using derivatives, **polynomial interpolation** uses a collection of sampled data points.

* 
**Given:** A set of $n+1$ points $(x_i, y_i)$ where $y_i = f(x_i)$ for $i = 1, \dots, n+1$.


* 
**Goal:** Find a polynomial $P(x)$ of degree $n$ such that $P(x_i) = y_i$ for all $i$.


* 
**Notation:** $\Pi_n$ denotes the collection of all polynomials over the reals with a degree of at most $n$.



#### **The Existence and Uniqueness Theorem**

For any $n+1$ arbitrary support points $(x_i, y_i)$, there exists a **unique** polynomial $P_n \in \Pi_n$ that interpolates those points. While there are many algorithms to find this polynomial, they all result in the same mathematical entity represented in different forms.

---

### **2. Lagrange Interpolating Polynomials**

Lagrange's method provides a straightforward way to prove existence by constructing the polynomial using specific basis functions.

#### **The Lagrange Basis**

For each point $i$, we define a basis polynomial $L_i(x)$ such that:


$$L_i(x) = \frac{\prod_{j \neq i} (x - x_j)}{\prod_{j \neq i} (x_i - x_j)}$$

* 
**Property:** $L_i(x_j) = 1$ if $i = j$, and $L_i(x_j) = 0$ if $i \neq j$.


* 
**Degree:** Each $L_i(x)$ has a degree of $n$.


* **Construction:** The full interpolating polynomial is the weighted sum:


$$P(x) = \sum_{i=1}^{n+1} y_i L_i(x)$$


.



> 
> **Note:** This method is elegant because it is "zero" at all interpolation points except the one it is specifically scaling.
> 
> 

---

### **3. Newton’s Interpolation Formula**

While Lagrange is conceptually simple, Newton’s method is often preferred for computers and allows for "building" the polynomial one point at a time.

#### **The Newton Basis**

Newton uses basis polynomials $n_j(x)$ defined as:

* 
$n_0(x) = 1$.


* 
$n_j(x) = \prod_{i=1}^{j} (x - x_i)$ for $j > 0$.



The interpolating polynomial takes the form:


$$N(x) = a_0 + a_1(x-x_1) + a_2(x-x_1)(x-x_2) + \dots + a_n n_n(x)$$

.

#### **Divided Differences**

The coefficients $a_j$ are determined using **Newton’s divided differences**, a recursive algorithm.

* 
$a_0 = y_1$.


* 
$a_1 = \frac{y_2 - y_1}{x_2 - x_1}$.


* Higher-order coefficients follow a recurrence relation that removes points with each iteration.



---

### **4. Worked Example: Newton’s Method**

**Data Points:** $(1, 2), (2, 10), (3, 5)$.

| Step | Calculation | Result |
| --- | --- | --- |
| **1. Basis Polynomials** | <br>$n_0=1$, $n_1=(x-1)$, $n_2=(x-1)(x-2)$ |  |
| **2. Find $a_0$** | $N(1) = a_0 = 2$ | <br>$a_0 = 2$ 

 |
| **3. Find $a_1$** | $N(2) = 2 + a_1(2-1) = 10 \implies a_1 = 8$ | <br>$a_1 = 8$ 

 |
| **4. Find $a_2$** | $N(3) = 2 + 8(3-1) + a_2(3-1)(3-2) = 5$ | <br>$a_2 = -\frac{13}{2}$ 

 |

**Final Polynomial:**


$$N(x) = 2 + 8(x-1) - \frac{13}{2}(x-1)(x-2)$$

.

---

### **Comparison of Methods**

| Method | Advantages | Disadvantages |
| --- | --- | --- |
| **Lagrange** | Mathematically "neat"; easy to write down manually.

 | Difficult to update if a new point is added.

 |
| **Newton** | Computational efficiency; can add points incrementally.

 | Slightly more cumbersome to compute by hand.

 |
